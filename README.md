# NoteFlow

A modern, private, offline-first note-taking application built with Next.js.

## 🌐 Live Demo

**[https://tug-on-dev.github.io/noteflow-demo](https://tug-on-dev.github.io/noteflow-demo)**

## Features

- 📝 Rich text editing with TipTap (bold, italic, headings, lists, code blocks, and more)
- 📁 Organize notes into notebooks
- 🏷️ Tag notes for quick filtering
- 🗑️ Trash & restore deleted notes
- 📌 Pin important notes
- 🔍 Full-text search
- 🌙 Dark/light/system theme
- 💾 Fully offline — all data stored in `localStorage`
- 🌐 Available in English, French, Italian, and Arabic (with RTL support)
- 📤 Export notes as Markdown or plain text

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This app is automatically deployed to [GitHub Pages](https://tug-on-dev.github.io/noteflow-demo) on every push to `main` via GitHub Actions.
